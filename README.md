# For loop

```go
	for x := 0; x < 10; x++ {
		fmt.Println(x)
	}

    y := 0
    for y < 10 {
        fmt.Println(y);
        y++;
    }

    # Same as while loop
    for {
        fmt.Println("This is an infinite loop");
    }
```
